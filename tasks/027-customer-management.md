# Task 027: 거래처 관리 기능 구현

## 상태
- [x] ✅ 완료

## 개요
거래처(고객사) 정보를 등록/수정/삭제하고 담당자를 관리하는 기능을 구현한다. 견적서 작성 시 거래처를 선택하면 수신자 정보가 자동 삽입된다. F020 구현.

## 관련 PRD 기능
- [F020] 거래처 관리

## 관련 파일
- `app/(dashboard)/customers/page.tsx` — 거래처 관리 페이지
- `app/api/customers/route.ts` — 거래처 CRUD API
- `app/api/customers/[id]/contacts/route.ts` — 담당자 API
- `lib/types/index.ts` — Customer, CustomerContact 타입
- `components/customers/` — 거래처 관련 컴포넌트
