## 구현 단계
- [ ] ExcelJS를 활용한 엑셀 템플릿 동적 생성 API 구현
  - 현재 DB 카테고리 목록 조회 → 드롭다운 유효성 검사 시트 포함
  - 필수 컬럼: 카테고리명, 모델명, 제조사, 리스트가, 시장가, 원가, 공급가
  - 선택 컬럼: 스펙 필드 (key=value 형식)
- [ ] 엑셀 파일 파싱 서비스 구현
  - .xlsx 파일 읽기 (ExcelJS)
  - 행별 유효성 검증 (필수 필드, 카테고리 존재 여부, 가격 양수)
  - 중복 부품 감지 (model_name + manufacturer + category)
- [ ] 엑셀 업로드 API 구현 (`POST /api/parts/excel-upload`)
  - 중복 처리 옵션 파라미터 (skip/overwrite)
  - 원가 AES-256 암호화 일괄 처리
  - 트랜잭션 기반 일괄 INSERT (성공분만 커밋)
  - excel_upload_logs 이력 저장
  - 업로드 시 가격 이력 자동 기록 (change_type: 'excel_upload')
- [ ] 업로드 결과 리포트 생성 (성공/실패 행 수, 오류 사유)
- [ ] 오류 행 엑셀 다운로드 API 구현
- [ ] 부품 관리 페이지 UI와 연동 (Task 008에서 만든 업로드 영역)
