# Task 035: 실판매 분석 대시보드

## 개요

낙찰 이력 페이지에 "실판매 분석" 탭을 추가하여, 견적 정확도 추이, 예상 vs 실제 마진 비교, 항목별 금액 차이, 과대/과소 견적 비율 등을 차트와 테이블로 시각화합니다. 거래처별 견적 정확도 랭킹도 제공합니다.

## 관련 기능

- **F022**: 견적 vs 실판매 비교 — 견적 정확도, 마진 편차, 카테고리별 분석 차트
- **F009**: 낙찰 이력 대시보드 — 기존 대시보드에 탭 추가

## 현재 상태

- 낙찰 이력 페이지(`/bid-history`) 존재
- 기존 대시보드: 낙찰률, 월별 추이, 실패 사유 분포 차트 존재
- Recharts 라이브러리 사용 중
- 실판매 분석 관련 UI 없음

## 수락 기준

- [ ] 낙찰 이력 페이지에 "실판매 분석" 탭이 추가됨
- [ ] 견적 정확도 추이 라인 차트가 표시됨 (월별)
- [ ] 예상 vs 실제 마진 스캐터 차트가 표시됨
- [ ] 항목별 금액 차이 바 차트가 표시됨 (카테고리별)
- [ ] 과대/과소 견적 비율 도넛 차트가 표시됨
- [ ] 거래처별 견적 정확도 랭킹 테이블이 표시됨
- [ ] 기간 필터(1/3/6/12개월)가 동작함
- [ ] `tsc --noEmit` + `npm run build` 통과

### Step 3: 차트 컴포넌트 구현

  - X축: 월별, Y축: 정확도 (%)
  - 기준선: 100% (정확)
  - 색상: 정확도 95% 이상 초록, 90~95% 노랑, 90% 미만 빨강
  - 데이터 포인트 hover 시 상세 표시 (건수, 평균 정확도)

- [ ] `components/actual-sales/charts/margin-scatter-chart.tsx` (신규)
  - **예상 vs 실제 마진 (스캐터 차트)**
  - X축: 예상 마진율 (%), Y축: 실제 마진율 (%)
  - 기준선: y=x (예상=실제)
  - 각 점: 개별 견적 건 (hover 시 견적번호, 거래처, 금액 표시)
  - 색상: 기준선 위(실제>예상, 초록), 기준선 아래(실제<예상, 빨강)

- [ ] `components/actual-sales/charts/category-difference-chart.tsx` (신규)
  - **항목별 금액 차이 (바 차트)**
  - X축: 부품 카테고리, Y축: 평균 금액 차이 (원)
  - 양수(과소 견적, 빨강 바), 음수(과대 견적, 파랑 바)
  - 카테고리별 견적 건수 라벨 표시

- [ ] `components/actual-sales/charts/estimate-ratio-chart.tsx` (신규)

  - 각 세그먼트 클릭 시 해당 견적 목록 표시 (선택적)
