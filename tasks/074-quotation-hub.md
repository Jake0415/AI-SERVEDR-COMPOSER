# Task 074: 견적 생성 허브 페이지

## 상태: 완료

## 관련 Feature
- 견적 생성 3가지 시나리오 통합 진입점

## 의존성
- 없음 (최우선 구현)

## 설명
기존 `/quotation` 페이지(견적안 비교 화면)를 **3가지 견적 생성 시나리오 선택 허브**로 재설계한다.

### 견적 생성 3가지 방법
1. **RFP 기반 견적**: RFP 문서(PDF/HWP/DOCX) 업로드 → AI 파싱 → 3가지 견적안
2. **엑셀 업로드 견적**: 서버 구성 엑셀 → 파싱 → 3가지 견적안
3. **AI 대화형 견적**: 자연어/가이드 모드 → AI 사양 결정 → 3가지 견적안

### 부가 기능
- **템플릿에서 시작**: 저장된 서버 구성 재활용 (Task 051)

## 수정 대상 파일
- `app/(dashboard)/quotation/page.tsx` — 허브 페이지로 재설계
- `components/layout/sidebar.tsx` — 메뉴 정리

## 사이드바 메뉴 변경
