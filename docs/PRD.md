# AI-SERVER-COMPOSER PRD — RFP 기반 인프라 견적서 산출 (프로덕션)

## 핵심 정보

**목적**: RFP 문서를 AI로 분석하여 인프라 부품을 자동 조합하고, 수익성/규격충족/성능향상 3가지 견적안을 산출하여 인프라 견적서 작성 시간과 비용을 혁신적으로 줄인다. 다양한 회사에 배포하여 실제 운영 가능한 프로덕션 수준의 솔루션.
**목표 수준**: 다수 회사에 배포하여 일상 업무에 사용 가능한 프로덕션 운영 품질
**사용자**: 서버/IT인프라를 조합하여 납품하는 사내 영업담당, 엔지니어, 관리자 (사내 전용 솔루션)

## 사용자 여정

```
[최초 설치 시]
1. 루트(/) 접속 → DB에 사용자 0명 감지 → 초기 설정 페이지로 자동 리디렉션
   ↓ [슈퍼어드민 계정 생성: 이메일/비밀번호/이름/전화번호/부서명 + 회사정보]
2. 로그인 페이지 → 슈퍼어드민으로 로그인
   ↓
3. 대시보드 (슈퍼어드민)
   ├─ 사용자 관리 → 관리자/멤버 추가
   ├─ 부품 관리 → 부품 DB 초기 등록
   └─ 이후 일반 업무 플로우

[일반 접속 시]
1. 루트(/) 접속 → 로그인 페이지
   ↓ [이메일/비밀번호 입력 → 로그인]
2. 대시보드 (역할별 메뉴 표시)
   ├─ [슈퍼어드민] → 모든 기능 + 사용자 관리
   ├─ [관리자] → 모든 견적 기능 + 부품 관리
   └─ [멤버] → 본인 견적 업무만
   ↓

3. 부품 관리 페이지 (관리자 이상)
   ↓ [부품 등록/원가 입력]

4. RFP 업로드 페이지
   ↓ [PDF/HWP/DOCX 업로드 → AI 파싱 → 요구사항 추출 확인]

5. 견적 생성 허브 페이지 (거래처 선택 → 견적 방법 선택)
   ├─ [Step 1] 거래처 선택: 의뢰 회사 검색/선택 또는 신규 등록 (필수)
   ├─ [Step 2] 견적 방법 선택 (거래처 선택 후 활성화)
   │   ├─ [방법 1] RFP 기반: PDF/HWP/DOCX 업로드 → AI 파싱 → 3가지 견적안 자동 생성
   │   ├─ [방법 2] 엑셀 업로드: 서버 구성 엑셀 → 파싱 → 3가지 견적안 생성
   │   ├─ [방법 3] AI 대화형: 자연어/가이드 모드 → AI가 사양 결정 → 3가지 견적안 생성
   │   └─ [부가] 템플릿에서 시작: 저장된 서버 구성 재활용
   └─ 선택한 거래처 정보가 모든 하위 페이지에 배너로 표시됨
   ↓ [마진 시뮬레이션 → 견적 확정]

6. 견적 이력 페이지
   ↓ [견적서 출력(PDF/Excel), 낙찰/실패 결과 기록]

7. 낙찰 이력 페이지
   └─ [낙찰률 확인, 경쟁력 분석 대시보드]
```

### F046: RFP 기반 서버 구성 워크플로우 (F005 하위 기능)

> ⚠️ 독립적인 견적 생성 방식이 아닌 **시나리오 1(RFP 기반 견적)**의 선택적 세부 조정 단계로 재분류됨

- RFP 파싱 결과에서 서버 장비 리스트 자동 추출
- 서버별 자동/수동 구성 선택
- 6단계 위저드: 섀시→CPU→메모리→스토리지→확장카드→PSU
- 호환성 실시간 검증
- 3가지 전략 비교 후 견적 확정

### F047: 엑셀 업로드 견적 생성 (신규)

- 규격 엑셀 템플릿에 서버 구성(서버명, 수량, CPU/메모리/스토리지 사양)을 입력하여 업로드
- 엑셀 파싱 → ParsedServerConfig로 변환 → 기존 견적 엔진 연동하여 3가지 견적안 생성
- 상세 부품 지정 시트(선택적): 특정 부품 모델을 직접 지정
- 기존 부품 엑셀 업로드(F001)와 별도 — 부품 DB 등록이 아닌 견적 생성 목적
- 관련 페이지: 견적 생성 허브 → 엑셀 업로드

## 기술 스택 (최신 버전)

### 프론트엔드 프레임워크

- **Next.js 15** (App Router) - React 풀스택 프레임워크
- **TypeScript 5.6+** - 타입 안전성 보장
- **React 19** - UI 라이브러리 (최신 동시성 기능)

### 스타일링 & UI

- **TailwindCSS v4** (CSS 기반 설정, tailwind.config 없음) - 유틸리티 CSS 프레임워크
- **shadcn/ui** (New York 스타일) - 고품질 React 컴포넌트 라이브러리
- **Lucide React** - 아이콘 라이브러리
- **Recharts** - 대시보드 차트 라이브러리

### 폼 & 검증

- **React Hook Form 7.x** - 폼 상태 관리
- **Zod** - 스키마 검증 라이브러리

### 백엔드 & 데이터베이스

- **Supabase** - BaaS (인증, 데이터베이스, 스토리지, RLS)
- **PostgreSQL** - 관계형 데이터베이스 (Supabase 포함, RLS로 멀티테넌시 구현)

### AI & 문서 처리

- **OpenAI GPT-4o** - RFP 문서 파싱 및 요구사항 추출
- **pdf-parse** - PDF 텍스트 추출
- **mammoth** - DOCX 텍스트 추출

### 파일 출력

- **jsPDF** - PDF 견적서 생성
- **ExcelJS** - Excel 견적서 생성

### 배포 & 호스팅

- **Vercel** (서울 리전 icn1) - Next.js 15 최적화 배포 플랫폼

### 패키지 관리
