# AI-SERVER-COMPOSER MVP PRD — RFP 기반 인프라 견적서 산출

## 핵심 정보

**목적**: RFP 문서를 AI로 분석하여 인프라 부품을 자동 조합하고, 수익성/규격충족/성능향상 3가지 견적안을 산출하여 인프라 견적서 작성 시간과 비용을 혁신적으로 줄인다.
**사용자**: 서버/IT인프라를 조합하여 납품하는 사내 영업담당, 엔지니어, 관리자 (사내 전용 솔루션)

---

## 사용자 여정

```
[최초 설치 시]
1. 루트(/) 접속 → DB에 사용자 0명 감지 → 초기 설정 페이지로 자동 리디렉션
   ↓ [슈퍼어드민 계정 생성: 이메일/비밀번호/이름/전화번호/부서명 + 회사정보]
2. 로그인 페이지 → 슈퍼어드민으로 로그인
   ↓
3. 대시보드 (슈퍼어드민)
   ├─ 사용자 관리 → 관리자/멤버 추가
   ├─ 부품 관리 → 부품 DB 초기 등록
   └─ 이후 일반 업무 플로우

[일반 접속 시]
1. 루트(/) 접속 → 로그인 페이지
   ↓ [이메일/비밀번호 입력 → 로그인]
2. 대시보드 (역할별 메뉴 표시)
   ├─ [슈퍼어드민] → 모든 기능 + 사용자 관리
   ├─ [관리자] → 모든 견적 기능 + 부품 관리
   └─ [멤버] → 본인 견적 업무만
   ↓

3. 부품 관리 페이지 (관리자 이상)
   ↓ [부품 등록/원가 입력]

4. RFP 업로드 페이지
   ↓ [PDF/HWP/DOCX 업로드 → AI 파싱 → 요구사항 추출 확인]

5. 견적 생성 페이지
   ↓ [AI가 3가지 견적안 자동 생성 → 마진 시뮬레이션 → 견적 확정]

6. 견적 이력 페이지
   ↓ [견적서 출력(PDF/Excel), 낙찰/실패 결과 기록]

7. 낙찰 이력 페이지
   └─ [낙찰률 확인, 경쟁력 분석 대시보드]
```

## 기능 명세

### 1. MVP 핵심 기능

| ID | 기능명 | 설명 | MVP 필수 이유 | 관련 페이지 |
|----|--------|------|---------------|------------|
| **F001** | 부품 DB 등록/관리 | 서버/인프라 부품 카테고리별 부품 정보와 원가/공급가를 등록 및 관리. 기본 14개 카테고리 제공. 카테고리 추가/삭제 가능. **엑셀 일괄 업로드**: 규격 템플릿으로 부품+가격 일괄 등록, 중복 처리(건너뛰기/덮어쓰기), 오류 행 리포트 다운로드. 엑셀 템플릿 동적 생성(현재 카테고리 반영) | 견적 생성의 기초 데이터 + 초기 DB 구축 필수 | 부품 관리 페이지 |
| **F002** | 부품 가격 관리 | 부품별 리스트가, 시장가, 원가(AES-256 암호화), 공급가를 등록하고 마진 자동 계산. **가격 변동 이력**: 수동 수정/엑셀 업로드 시 변경 전후 자동 기록. **가격 변동 시각화**: 부품별 가격 추이 차트(4가지 가격 시계열). **자동 스냅샷**: 매일 설정 시간에 전체 가격 스냅샷 저장(보관 기간 설정 가능) | 견적 산출의 핵심 데이터 + 가격 추세 분석 | 부품 관리 페이지 |
| **F003** | RFP 파일 업로드 및 이력 관리 | PDF, HWP, DOCX 형식의 RFP 문서를 업로드하고, 업로드 이력을 목록으로 관리. 기존 RFP 선택 시 재견적 생성 가능 | 견적 자동화의 시작점 + 문서 이력 추적 | RFP 업로드 페이지 |
| **F004** | RFP AI 파싱 | 업로드된 RFP에서 서버 사양 요구사항(CPU, 메모리, 스토리지, 네트워크 등)을 AI로 자동 추출 | 핵심 차별화 기능 | RFP 업로드 페이지 |
| **F005** | 3가지 견적안 자동 생성 | 수익성 중심안(마진 최대화), 규격 충족안(RFP 정확 매칭), 성능 향상안(10~30% 업스펙)을 자동 생성 | 핵심 비즈니스 가치 | 견적 생성 페이지 |
| **F006** | 마진 시뮬레이션 | 부품별 마진율 슬라이더 조절, 목표 마진 역산, 견적안 간 시나리오 비교 | 영업 의사결정 핵심 도구 | 견적 생성 페이지 |
| **F007** | 견적서 출력 | 나라장터 양식/자사 양식 선택, PDF/Excel 형식 다운로드 | 최종 산출물 제공 | 견적 이력 페이지 |
| **F008** | 낙찰 결과 기록 | 견적별 낙찰/실패 결과와 사유를 기록 | 데이터 축적으로 경쟁력 강화 | 견적 이력 페이지 |
| **F009** | 낙찰 이력 대시보드 | 낙찰률, 평균 마진율, 카테고리별 성과 등 경쟁력 분석 대시보드 | 영업 전략 개선 인사이트 | 낙찰 이력 페이지 |

## 페이지별 상세 기능

### 초기 설정 페이지

> **구현 기능:** `F010`, `F011` | **메뉴 위치:** 비로그인 > 초기 설정 (최초 1회)

| 항목 | 내용 |
|------|------|
| **역할** | 최초 설치 시 슈퍼어드민 계정 및 테넌트(회사) 생성. DB에 사용자가 0명일 때만 표시 |
| **진입 경로** | 루트(/) 접속 시 DB에 사용자 0명이면 자동 리디렉션 |
| **사용자 행동** | 슈퍼어드민 계정 정보 입력 (이메일/비밀번호/이름/전화번호/부서명), 회사 정보 입력 (회사명/사업자등록번호) |
| **주요 기능** | - 이메일(ID로 사용)/비밀번호 입력 및 유효성 검사<br>- 이름, 전화번호, 부서명 입력<br>- 회사명, 사업자등록번호 입력<br>- 테넌트 자동 생성 (데이터 격리 적용)<br>- 슈퍼어드민 역할 자동 부여<br>- **초기 설정 완료** 버튼 |
| **다음 이동** | 성공 → 로그인 페이지 (생성한 계정으로 로그인 유도), 실패 → 에러 표시 |

> **구현 기능:** `F012` | **메뉴 위치:** 메인 메뉴 > 대시보드 | **인증:** 필수

| 항목 | 내용 |
|------|------|
| **역할** | 로그인 후 메인 진입점, 현황 요약 및 빠른 액션 제공 (역할별 메뉴 표시) |

| **주요 기능** | - 최근 견적 목록 (최신 5건, 멤버는 본인 건만)<br>- 진행 중/완료/낙찰 건수 요약 카드<br>- 월간 낙찰률 요약<br>- **RFP 업로드** 바로가기 버튼<br>- **부품 관리** 바로가기 버튼 (관리자 이상)<br>- **사용자 관리** 바로가기 (슈퍼어드민만) |
| **다음 이동** | RFP 업로드 → RFP 업로드 페이지, 부품 관리 → 부품 관리 페이지, 견적 클릭 → 견적 생성 페이지 |

### 부품 관리 페이지

> **구현 기능:** `F001`, `F002` | **메뉴 위치:** 메인 메뉴 > 부품 관리 | **인증:** 필수

| 항목 | 내용 |
|------|------|
| **역할** | 서버 부품 DB 등록/수정/삭제 및 가격 정보 관리 |
| **진입 경로** | 좌측 메뉴 '부품 관리' 클릭, 대시보드 바로가기 |
| **사용자 행동** | 카테고리 선택, 부품 추가/수정, 원가/공급가 입력 |
| **주요 기능** | - 카테고리 탭/그룹 전환: **서버 부품**(CPU, 메모리, SSD, HDD, NIC, RAID, GPU, PSU, 메인보드, 섀시, HBA) / **네트워크·인프라**(스위치, 광 트랜시버, 케이블)<br>- **카테고리 추가**: 관리자 이상이 새 카테고리 생성 (카테고리명, 그룹, 스펙 필드 정의)<br>- **카테고리 삭제**: 하위 부품이 0건일 때만 삭제 가능. 부품이 있으면 삭제 차단 및 경고 표시<br>- 부품 추가/수정/삭제 (모델명, 제조사, 주요 사양)<br>- 가격 4단계 입력: 리스트가, 시장가, 원가(암호화 저장), 공급가<br>- 마진 자동 계산 표시 (공급가 - 원가)<br>- 부품 목록 테이블 (검색, 카테고리 필터) |
| **다음 이동** | 부품 등록 완료 → 부품 목록 유지, RFP 업로드 → RFP 업로드 페이지 |

| 항목 | 내용 |

| **주요 기능** | - **RFP 이력 목록** (날짜, 파일명, 파싱 상태, 연결된 견적 수)<br>- 이력에서 기존 RFP 선택 → 재견적 생성 가능<br>- 파일 업로드 영역 (PDF, HWP, DOCX 지원, 드래그앤드롭)<br>- AI 파싱 진행 상태 표시 (프로그레스 바)<br>- 추출된 요구사항 목록 표시 (서버 대수, CPU, 메모리, 스토리지, 네트워크 등)<br>- 추출 결과 수동 수정/보완 기능<br>- RFP 문서 삭제 기능<br>- **견적 생성하기** 버튼 |
| **다음 이동** | 견적 생성하기 → 견적 생성 페이지, 기존 RFP 선택 → 파싱 결과 표시, 취소 → 대시보드 |

### 견적 생성 페이지

| 항목 | 내용 |

| **진입 경로** | RFP 업로드 페이지에서 '견적 생성하기' 클릭 |
| **사용자 행동** | 3가지 견적안 비교 검토, 부품별 마진 슬라이더 조절, 목표 마진 역산, 최종 견적안 선택/확정 |
| **주요 기능** | - 3가지 견적안 탭 표시 (수익성 중심 / 규격 충족 / 성능 향상)<br>- 각 안별 부품 구성 테이블 (부품명, 수량, 원가, 공급가, 마진)<br>- 부품별 마진율 슬라이더 실시간 조절<br>- 목표 마진 금액 입력 → 부품별 가격 역산<br>- 견적안 간 총액/마진 비교 요약<br>- **견적 확정** 버튼 |
